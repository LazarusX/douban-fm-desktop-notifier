豆瓣电台桌面提醒助手
===

为豆瓣电台网页版开启桌面提醒功能的Chrome扩展。

用法
---
1. 登录豆瓣电台网页版（http://douban.fm）
2. 点击页面右上角的「打开提醒」
3. 在弹出的banner中选择「允许」

已知问题
---
1. 可能和其他豆瓣电台网页版的扩展或userscript产生冲突
2. 必须登录豆瓣帐号后才可开启提醒
